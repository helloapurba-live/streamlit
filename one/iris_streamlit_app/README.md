# Iris Streamlit App

This is an interactive machine learning application for classifying Iris flowers.

## Setup

1.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

2.  **Run the App**:
    ```bash
    streamlit run app.py
    ```

## Features

-   **Interactive Sliders**: Adjust Sepal and Petal measurements.
-   **Real-time Prediction**: See the predicted species and probability instantly.
-   **Visualizations**:
    -   Scatter plot comparing your input to the original dataset.
    -   Feature importance chart.
